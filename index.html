Mini iLovePDF Web

Juntar, dividir e comprimir PDFs direto no navegador.

Pronto para GitHub Pages

Juntar PDFs e imagens

Selecione PDFs e imagens. As imagens serão convertidas em páginas PDF e tudo será unido em um único arquivo.

Dividir PDF por página

Gera um arquivo ZIP com cada página separada em um PDF.

Comprimir PDF

Esta versão usa rasterização no navegador. É a alternativa viável para GitHub Pages. Quanto menor o DPI e a qualidade, menor o arquivo.
